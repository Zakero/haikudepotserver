Вы можете создать иконки с размерами 16px и 32px, открыв векторную иконку в Icon-O-Matic и экспортировав ее в виде файлов PNG 16x16px и PNG 32x32px. Наиболее простой способ открыть векторную иконку - открыть бинарный файл соответствующего приложения при помощи дополнения Tracker 'FileType' и дважды кликнуть по иконке в нем.

Иконка HVIF создается аналогичным образом посредством экспортирования в формат HVIF при помощи Icon-O-Matic.

Вы можее извлечь иконки с размерами 16px и 32px из приложения, не имеющего векторную иконку (к примеру, из программы для BeOS), при помощи инструмента для командной строки app2png, доступного на HaikuDepot.
