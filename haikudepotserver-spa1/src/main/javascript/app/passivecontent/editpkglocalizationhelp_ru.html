Локализация пакета - переведенные на тот или иной язык дополнительные значения полей, хранящиеся в пакете (версии пакета). Слева содержится список языков для локализации. Выберите язык для отображения полей с данными. Вы можете редактировать их и, после завершения данной процедуры для одного или более языков, нажать кнопку внизу страницы для сохранения данных. Выбор языка отображения того или иного поля (заголовок, краткое и подробное описание) основан на следующем порядке;

  1. Данные пакета для пользовательского языка
  2. Дополнительные данные пакета для пользовательского языка
  3. Данные пакета для английского языка
  4. Дополнительные данные пакета для английского языка
