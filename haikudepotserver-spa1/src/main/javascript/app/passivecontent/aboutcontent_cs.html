Účel

Tato webová aplikace je online nástroj pro práci se softwarovými balíčky Operační systém Haiku (Haiku). Webová aplikace má za cíl poskytovat následující funkce;

  • Internetový katalog balíčků.
  • Úložiště dalších dat souvisejících s balíčky, které nejsou součástí standardu formát balíčku — jako je;
    • Snímky obrazovky
    • Ikonografie
    • Lokalizace
    • Zpětná vazba od uživatelů
  • Uživatelské rozhraní pro manipulaci s dalšími daty.
  • Mechanismus pro prodej těchto dalších dat do HaikuDepot, desktopové aplikace pro správu balíčků.
