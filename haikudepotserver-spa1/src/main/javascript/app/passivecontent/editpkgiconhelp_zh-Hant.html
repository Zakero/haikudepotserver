您可以通過在 “Icon-O-Matic” 中打開一個矢量圖示，然後將其導出為 16x16 像素的 PNG 格式和 32x32 像素的 PNG 格式，從而創建 16 像素和 32 像素的圖示。最簡單的打開矢量圖示的方法是，使用 “FileType Tracker” 插件打開帶有該圖示的應用程式二進制檔案，然後在那裡雙擊該圖示。

HVIF 圖示檔案也是以相同的方式建立，也就是從 Icon - O - Matic 將圖示匯出為 HVIF 格式。

若應用程式沒有矢量圖示（例如舊版 BeOS 圖示），您可以使用從 Haiku 軟件倉庫（HaikuDepot）取得的命令列工具 app2png，從該應用程式中提取 16 像素和 32 像素的圖示。
