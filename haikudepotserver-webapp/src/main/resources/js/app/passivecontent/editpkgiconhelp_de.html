16px und 32px Icons können erzeugt werden, indem man ein Vektoricon in Icon-O-Matic öffnet und als 'PNG 16x16px' bzw. 'PNG 32x32px' exportiert. Das Vektoricon öffnet man am einfachsten, indem man die Anwendung mit dem Icon mit dem FileType Tracker Add-on öffnet und das Icon dort doppelklickt.

Das HVIF Icon erzeugt man auf die gleiche Weise, durch Export als 'HVIF' in Icon-O-Matic.

Man kommt an die 16px und 32px Icons von einer Anwendung ohne Vektoricon (also ein altes BeOS Icon) mit dem Kommandozeilen-Tool 'app2png' aus dem HaikuDepot.
